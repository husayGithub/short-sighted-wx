# 短视反思记录微信小程序

这是一个关于短视反思记录的微信小程序项目。

## 项目结构

```
├── app.js          # 小程序入口文件
├── app.json        # 小程序全局配置
├── app.wxss        # 小程序全局样式
├── pages/          # 页面目录
│   ├── index/      # 首页
│   └── history/    # 历史页面
└── utils/          # 工具函数
    ├── request.js  # 网络请求工具
    └── util.js     # 通用工具函数
```

## 开发环境

- 微信开发者工具
- Node.js (如果需要)

## 开发说明

1. 使用微信开发者工具打开项目
2. 导入项目到开发者工具
3. 开始开发调试
